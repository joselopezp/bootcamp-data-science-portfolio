# Decisions Log

> Record of key analytical and methodological decisions made during
> the project. Essential for reproducibility and stakeholder transparency.

## How to Use This Log

For each significant decision, record:
- **What** was decided
- **Why** (business or technical rationale)
- **Alternatives** considered
- **Impact** on the analysis
